⚡ Quick Wins You Can Try Today

  • 1 Enable Copilot Memory in VS Code: set github.copilot.chat.copilotMemory.enabled = true. Copilot remembers project context across sessions — no more re-explaining architecture.
  • 2 Add an AGENTS.md file at your repo root describing your stack and conventions. VS Code and JetBrains pick it up automatically in agent sessions.
  • 3 Try Copilot CLI plan mode: install with npm i -g @github/copilot-cli, run copilot, then press Shift+Tab to get a full implementation plan before any code is written.
